## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. For calculator integration, replace the placeholder in the calculator section with your iframe

## Calculator Integration

The calculator section is ready for iframe integration. Simply replace the `.calculator-placeholder` div with your calculator iframe:

```html
<iframe src="your-calculator-url" width="100%" height="600px"></iframe>
```
